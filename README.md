# К2 Платформа

Комплексное SaaS-решение для балконных компаний: 3D-визуализация, остекление, калькулятор материалов, CRM, сметы, склад.

## Быстрый старт

```bash
# 1. Установить зависимости
npm install

# 2. Создать файл переменных окружения
cp .env.example .env
# Заполнить VITE_SUPABASE_URL и VITE_SUPABASE_ANON_KEY

# 3. Запустить Supabase миграцию (если используешь Supabase CLI)
# supabase db push

# 4. Запустить dev-сервер
npm run dev
```

Приложение откроется на http://localhost:5173

## Структура проекта

```
src/
  modules/
    core/           — Авторизация, лейаут, роутинг, настройки
    crm/            — CRM: клиенты, канбан-доска заказов
    visualizer3d/   — 3D-визуализатор (Three.js, Этап 4)
    glazing/        — 2D-конструктор остекления + расчёт
    calculator/     — Калькулятор материалов (8 режимов)
    estimates/      — Генерация смет PDF/Excel
    warehouse/      — Складской учёт
    billing/        — Тарифы, подписки, ЮКасса
  shared/           — Общие компоненты, типы, утилиты
  store/            — Zustand сторы
  lib/              — Supabase клиент
supabase/
  migrations/       — SQL-миграции
  functions/        — Edge Functions (ЮКасса webhook и др.)
```

## Стек

- **Фронтенд:** React 19 + TypeScript + Vite + Tailwind CSS
- **Стейт:** Zustand
- **Бэкенд:** Supabase (PostgreSQL + Auth + Edge Functions)
- **3D:** Three.js / React Three Fiber (Этап 4)
- **Платежи:** ЮКасса (Этап 5)
